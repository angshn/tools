## 用法
**yauth**

```shell
# login
$ python yauth -l yourstudentnumber
>input your password hear
Login successfully.


# logout
$ python yauth -q
下线成功

# 查看登录信息
$ python yauth -t
Logged!
Current user is xx.
IP is 10.20.xx.xx.
ISP is 中国yy
```
